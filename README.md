# Panda Utils

## Usage

Install [extension](https://marketplace.visualstudio.com/items?itemName=EdwinKofler.vscode-panda-utils)

Snippets for

- Basalt
- Desktop Files
- EditorConfig
- ESLint
- Prettier
- `settings.json`
- Stylelint
- Webpack

Syntax highlighting for

- AWK
- Desktop Files
- dircolors
- i3
- JSON5
- Ninja
- gettext
- sxhkdrc
- systemd
- terminfo
- Xresources
